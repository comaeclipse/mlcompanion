# Project Overview

## Project Name
MLCompanion

## Purpose
MLCompanion is a Learning Hub for Socialism and Marxism - an educational platform with video library management, user authentication, and structured learning paths.

## Key Features
- Video library management with YouTube integration
- User authentication (session-based with email/password)
- YouTube metadata auto-fetching
- Tag-based video categorization
- User-owned video management (create, edit, soft delete)
- Soft delete pattern (videos marked unpublished, not deleted)
- Structured learning paths (planned)

## Deployment
- Platform: Vercel (serverless functions)
- Database: Neon (PostgreSQL)
- Environment: Production

- Core video management: Complete
- Authentication: Complete
